## Section 03：左側工具列

### 選取群組

#### ✅ 選取（V）

**操作方式**
- 點選標注物件以選取；點選空白處取消選取
- 空白處拖曳可框選多個物件
- Shift＋點選可逐一加選／取消物件

**選取單一物件時**
顯示該物件的所有子屬性（依物件類型而定），包含顏色、線寬、透明度、大小、陰影等屬性

**選取複數物件時（多選模式）**
顯示對齊工具列：

| 按鈕 | 功能 |
|------|------|
| 齊左 | 對齊最左邊物件的左緣 |
| 水中 | 水平置中 |
| 齊右 | 對齊最右邊物件的右緣 |
| 齊上 | 對齊最上方物件的上緣 |
| 垂中 | 垂直置中 |
| 齊下 | 對齊最下方物件的下緣 |
| 水均 | 水平均分間距 |
| 垂均 | 垂直均分間距 |
| 對齊中線 | 對齊畫布中心線 |
| ↔ | 水平鏡射 |
| ↕ | 垂直鏡射 |

#### ✅ 框型選取（M）

**操作方式**
- 拖曳框選像素區域
- 框選後可將該範圍的圖片內容複製貼上，成為新的物件

**屬性**
無可調屬性

---

### 繪圖群組

#### ✅ 筆型（P）

**操作方式**
- 在畫布上拖曳自由手繪平滑曲線

**屬性**
顏色、線寬（px）、線條樣式（實線／虛線）、端點樣式（無／箭頭／圓點／方形）、筆畫描邊色、透明度、陰影

#### ✅ 線條（L）

**操作方式**
- 拖曳繪製直線；Shift＋拖曳鎖定 90° 直角
- 選取後拖曳中間控制點可轉為貝茲曲線

**屬性**
顏色、線寬、線條樣式（實線／虛線）、起端樣式（無／箭頭／圓點）、末端樣式（無／箭頭／圓點）、直角折線模式、水平鏡射、垂直鏡射、筆畫描邊色、透明度、陰影

#### ✅ 矩形框（R）

**操作方式**
- 拖曳繪製外框；Shift＋拖曳鎖定正方形／圓形

**形狀**
矩形框 / 橢圓框（橢圓同樣支援 Shift 鎖定圓形）

**屬性**
顏色、線寬、圓角半徑（矩形適用，0–100%）、線條樣式（實線／虛線）、描邊色、位移、水平鏡射、垂直鏡射、透明度、陰影

#### ✅ 色塊（B）

**操作方式**
- 拖曳繪製填色形狀；Shift＋拖曳鎖定正方形／圓形

**形狀**
矩形色塊 / 橢圓色塊

**屬性**
填色（純色／漸層，漸層可設定兩端顏色）、圓角半徑（僅矩形色塊適用，0–100%）、邊框（寬度＋顏色＋線條樣式）、透明度、陰影

### 標注群組

#### ✅ 文字（T）

**操作方式**
- 點選畫布放置文字框；Enter 換行；Shift＋Enter 確認；雙擊已有文字可重新編輯

**屬性**
顏色、字型（PingFang TC / Heiti TC / Songti TC / Kaiti TC / Helvetica Neue / Georgia / Verdana / Impact / Menlo）、字級（8–400px，提供快捷預設值）、樣式（粗體／斜體／底線／刪除線）、對齊（左／中／右）、文字描邊（可自行輸入數值，提供快捷下拉選單）、文字背景色＋透明度、陰影

#### ✅ 編號（N）

**操作方式**
- 點選畫布放置自動遞增數字標記

**屬性**
尺寸（小／標準／大）、顏色、樣式（圓點／圓圈／實心圓／羅馬數字／中文圓括號／中文圓圈）、描邊粗細、描邊顏色、陰影、重置編號

#### ✅ 符號印章（U）

**操作方式**
- 點選畫布放置符號；從浮動面板挑選符號類別與圖案

**符號分類**
形狀、字母（圓圈體／全形／粗體／粗斜體／花體）、箭頭、其他（標記／貨幣／數學／技術符號）

**屬性**
顏色、大小（px）、陰影

#### ✅ 馬賽克／模糊（X）

**操作方式**
- 拖曳框選區域，立即套用遮蔽效果
- 套用後可切換至選取工具調整遮蔽範圍大小，或刪除遮蔽物件，以補償自動辨識遮蔽的不足

**屬性**
模式（馬賽克／模糊）、馬賽克區塊尺寸（4／8／16／32 px）、模糊半徑（4／8／16／24 px）

### 隱私與遮蔽群組

#### ✅ OCR 文字辨識（G）

**操作方式**
- 拖曳選取區域，自動辨識範圍內的文字（支援中英文）
- 辨識結果顯示於浮動面板，提供「複製文字」按鈕

**屬性**
無可調屬性

#### ✅ 隱私遮蔽（K）

**操作方式**
- 按 K 對全圖掃描；或拖曳指定範圍後掃描
- 自動偵測並遮蔽敏感資料：身分證字號、統一編號、信用卡號、電話、Email、IPv4／IPv6、API Token 等
- 套用後可切換至選取工具微調或刪除遮蔽物件

**屬性**
模式（馬賽克／模糊）、馬賽克區塊尺寸（8／16／24／32 px）、模糊半徑（4／8／16／24 px）

> ⚠️ EN 版翻譯注意：偵測範圍中「身分證字號」「統一編號」為臺灣特有格式，英文版描述應對應為 National ID / Tax ID，避免直譯造成誤解。

### 畫布操作群組

#### ✅ 放大（⌘=）

**操作方式**
- 以游標為中心放大畫面，範圍 25%–400%

**屬性**
縮放比例快捷下拉選單：25% / 50% / 75% / 100% / 150% / 200% / 300% / 400%

---

#### ✅ 縮小（⌘-）

**操作方式**
- 以游標為中心縮小畫面，範圍 25%–400%

**屬性**
縮放比例快捷下拉選單：25% / 50% / 75% / 100% / 150% / 200% / 300% / 400%

---

#### ✅ 適合視窗（⌘0）

**操作方式**
- 按下後立即將畫面縮放至 100% 並 fit 視窗

**屬性**
無可調屬性

### 版面功能群組

#### ✅ 裁切（C）

**操作方式**
- 拖曳建立裁切框，8 個控制點可調整範圍
- 方向鍵微調 1px，Shift＋方向鍵 10px
- 雙擊畫布或點選確認按鈕完成裁切

**屬性**
確認裁切按鈕、取消按鈕

#### ✅ 調整大小（S）

**操作方式**
- 開啟 popup 視窗（右上角有關閉按鈕）
- 輸入目標寬度，高度依比例自動計算
- 所有標注物件同步等比縮放
- 範圍：1–8192 px

**屬性**
寬度輸入（px）、高度（唯讀，自動計算）、套用按鈕、取消按鈕

#### ✅ 延伸畫布（E）

**操作方式**
- 開啟 popup 視窗（右上角有關閉按鈕）
- 選擇延伸方向（九宮格方向盤）
- 輸入延伸距離（px）
- 背景固定白色

**屬性**
延伸方向（九方向）、延伸距離（px）、延伸後尺寸預覽（px）、確認延伸按鈕、取消按鈕

#### ✅ 疊入圖片（O）

**操作方式**
- 點選後開啟檔案選擇器，選取圖片疊加於畫布上
- 可拖曳移動位置，8 個控制點縮放
- 作為浮動物件持續存在，拖曳匯出時自動合併至底圖
- 支援格式：JPG、PNG、WebP、GIF（第一幀）、SVG

**屬性**
無可調屬性

#### ✅ 一鍵套版

**操作方式**
- 開啟模板面板，選擇漸層背景樣式
- 目前提供 6 款：Apple 紅／橙／黃／綠／藍／紫
- 支援撤銷還原

**屬性**
留白（2–30%）、圓角（0–10）、陰影（0–10）

**社群媒體尺寸預設**
LinkedIn（1200×627）、Instagram 1:1（1080×1080）、Instagram 4:5（1080×1350）、X（1600×900）
